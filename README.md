# cal-server-client  

본 프로젝트는 **C 언어로 구현한 소켓 통신 기반 계산기 프로그램**으로, 클라이언트의 계산 요청을 서버가 처리하여 결과를 반환하는 구조를 갖고 있습니다.  
서버는 **TCP 프로토콜**을 통해 클라이언트의 연결을 수락하고, `fork()` **함수**를 이용해 **각 세션을 독립 프로세스로 병렬 처리**함으로써 다중 접속 환경에서도 안정적인 통신이 가능하도록 설계되었습니다.  
이를 통해 **네트워크 소켓의 동작 원리**, **프로세스 기반 병렬 처리 구조**, 그리고 **시스템 프로그래밍의 핵심 개념**을 직접 구현하며 실습적으로 이해할 수 있었습니다.  

---

### ⚙️ 주요 특징
- **C 언어 기반** TCP 소켓 통신 구조 구현  
- **`fork()` 함수**를 통한 멀티프로세스 병렬 처리  
- 서버–클라이언트 간 **요청(Request) / 응답(Response)** 구조 설계  
- 네트워크 바이트 오더(`htonl()`, `ntohl()`) 변환 처리  

---

### ⚠️ 주의사항
- 본 프로젝트는 **Linux 환경**에서만 정상적으로 동작합니다.  
  <sub>Windows에서는 `unistd.h`, `arpa/inet.h` 등의 헤더 파일이 지원되지 않습니다.</sub>
  
- 컴파일 시 반드시 **GCC 컴파일러**를 사용해야 합니다.  
  ```bash
  gcc server.c -o server
  gcc client.c -o client
